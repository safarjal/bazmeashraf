Urdu Date and Time Display to covert date and time into Urdu Language according to English calendar.


== Description ==

This plugin will display all date, time and numbers of WordPress website into Urdu Language. Urdu date and time will be shown according to English calendar only.

= Features =
* All dates will be displayed in Urdu numbers.</br>
* All time will be displayed in Urdu numbers.</br>
* Name of the seven week days will be displayed in Urdu Language.</br>
* Name of the months will be displayed in Urdu Language.</br>
* Display your Dashboard numbers in Urdu.

= Credits =
* Developer : Imran Ali Mughal (http://www.imughal.com)</br>



== Installation ==

1. Download this plugin.

2. Upload the zip file of this plugin into `wp-content/plugins/` directory.

3. Activate the plugin through the plugins menu.

All done! Enjoy Urdu Date and Time Display in your WP Weblog.
